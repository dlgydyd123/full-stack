keydown		onkeydown	키가 눌렸을 때

doubleclick	ondoubleclick	요소가 더블클릭되었들 때
mouseover	onmouseover	요소 위에 마우스포인터가 올라왔을 때
mouseleave 	onmouseleave	요소로부터 마우스포인터가 벗어났을 때

submit		onsubmit		form요소의 제출버튼이 클릭되었을 때
reset 		onreset		form요소의 리셋버튼이 클릭되었을 때

서버에 올라간 웹페이지는 실행중인 폴더 안에 있는 코드이다.
코드가 수정되면 웹 브라우저에서 보이는 페이지 내용도 자동으로 수정된다.

지킬 프로젝트 기본 구조

_config.yml 	웹 페이지의 환경설정과 정보를 보관
_data/		웹 페이지에서 활용할 데이터를 저장
_includes/	- 웹 페이지에 포함시킬 html 파일
 		- 주로 개별 페이지들이나 단락 저장
_layouts/		웹 페이지의 뼈대를 결정
_posts/		블로그의 게시물을 저장
_sass/		CSS의 변종인 sass(scss)파일들을 저장
_site/		- index.html 역활을 수행할 파일 저장
		- 혹은 규격에 맞춘 마크다운 파일도 사용가능

_config.yml
